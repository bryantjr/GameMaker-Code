Villains of the Game Maker Universe

Supreme Villain

Doctor Aakom is the main antagonist of the Game Maker universe. Like Doctor Eggman and Doctor Wily, he seeks to conquer and control all digital, video game, and anime worlds through corrupted data and forbidden creations.

Doctor Aakom’s Core Creations

  • Game Overload – Neo Metal Sonic (Final Fortress)
  • Game Data – Shadow.exe (Space Colony)
  • Game Creator – Silver.exe (Crisis City)
  • Game Chaos – Fleetway Super Sonic (Flame Core)
  • Game Virus – Mephiles (Dark Desert)
  • Game Error – Black Doom (Digital Dimension)
  • Game Glitch – Infinite Game (Scrap Brain Zone)
  • Game Auto – Metal Sonic (Chemical Planet)
  • Game Corruption – Chaos 0 (Labyrinth Zone)
  • Game Pixel – Nazo (Mushroom Hill)
  • Game Code – Aaron (Ice Cap Zone)
  • Game Relay – Scourge the Hedgehog (Casino Night)
  • Game Execution – Erazor Djinn (Rusty Ruin Zone)
  • Game Crusher – Dark Queen & King Arthur (Mystic Mansion)
  • Game Surge – Imperator IX (Star Light Zone)

Ultimate Weapon

  • Shadow Mewtwo – Created using the combined DNA of: Bowser, Ganondorf, and Sigma. Known as the “Shadow of the Emperor,” Shadow Mewtwo returns from death as Doctor Aakom’s most dangerous weapon.

  • They may be sealed, defeated, or corrupted

  • Doctor Aakom is the ultimate authority
